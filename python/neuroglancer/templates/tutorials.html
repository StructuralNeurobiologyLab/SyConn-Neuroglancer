This is pretty straightforward. Just select an acquisition and version from the dropdowns and click the Submit button to launch the viewer.

Currently, only the j0251 acquisition with versions rag_flat_Jan2019_v2 and rag_flat_Jan2019_v3 is available. More datasets will be added soon!

There are icons at the top right of the page that can be clicked to open the different panels.

  • ? : help panel (displays all the key and mouse bindings for the viewer)
  • : layer settings panel (displays the data source, rendering options, segment list and annotations)
  • : layer list panel (displays all the layers for archiving or toggling visibility)

Neuroglancer is easy to use with a mouse but the keyboard shortcuts come in handy to quickly jump through layers. For example, layers can be toggled by pressing layer_number on the keyboard (not numpad) and layer side panel can be opened by hitting Ctrl+layer_number or Ctrl+left_click

Neuroglancer has four viewports by default. Starting from left, in the clockwise-direction, these are the XY, XZ and YZ cross-sectional views and the 3D view. A segment (from the segmentation layer) is a coloured area visible in the three cross-sectional views that can be clicked to render its cell objects (if available).

  • left-click to select the segment
  • double-click to request the cell objects (skeleton, cell mesh and organelle meshes)
  • x to clear all segments

With the layer toggling, one can choose to see only one cell object or a combination thereof.

One way to select a segment is to click in the cross-sectional view. The other is to open the layer side panel and head to the Seg. tab. At the bottom is a list of all the segments with the corresponding segment ids. Any segment can be selected by checking the box left to the id column. This segment list can be modified based on the properties—cell type, mitochondria count within a cell, size of the cell.

For example, segment ids of the HVC cell type can be selected by clicking the left to #HVC. Moreover, segment ids of all the cell types except HVC can be selected by clicking the left to #HVC. Multiple cell types can be excluded.

Filtering by the mitochondria count and cell size can be performed by clicking the numerical properties dropdown. Checking the property box allows to display the property value as a column in the segment list. The slider bar below the property can be adjusted to set the value.

All the properties can be modified simultaneously according to one's needs.
