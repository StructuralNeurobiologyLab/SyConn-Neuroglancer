Welcome to the SyConn Neuroglancer Viewer!

This is an adaptation of the Google's Neuroglancer viewer to visualize Knossos Datasets and leverage the inference and analysis tools provided by SyConn. We support rendering of segmentation and raw datasets (3D volumes), requesting cell objects - skeletons, cell mesh, organelle meshes, and filtering segments based on properties e.g cell type, mitochondria count, ssv size and largest synaptic partner. This project is a work in progress. Feedbacks are always welcome :)
